## Descrição

Este projeto tem como finalidade servir como base para a implementação de testes automatizados no processo seletivo de QA da WeFit.

## Para rodar o projeto:

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Rota para acessar o Swagger: 
```bash
http://[host]/api
```
